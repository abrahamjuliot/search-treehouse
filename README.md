# search-treehouse
Node App: Search Treehouse Users by subject profile name(s) (using Treehouse API)

Run node app.js [subject] [username1, username2, ...]

Subjects include:

- JavaScript
- iOS
- CSS
- HTML
- Development Tools
- PHP
- Ruby
- Design
- Python
- Business
- Databases
- Digital Literacy
- WordPress
- Java
- Android
- C#
- Go
- Game Development
- Data Analysis
- APIs
- Security
